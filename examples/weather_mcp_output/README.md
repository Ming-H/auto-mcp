# weather-mcp

MCP server for Weather API

## Installation

```bash
pip install weather-mcp
```

## Usage

Run the server:

```bash
weather-mcp
```

Or with MCP inspector:

```bash
npx @modelcontextprotocol/inspector weather-mcp
```

## Tools

This MCP server provides the following tools:

- `get_current_weather`: Get current weather
- `get_weather_forecast`: Get weather forecast


## API Base URL

https://api.weather.example.com
